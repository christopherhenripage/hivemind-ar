# Brand Icons Required

The following icon files are needed for complete favicon and PWA support.
Generate these from your HiveMind logo.

## Favicons (Browser tabs)
- `favicon-16.png` - 16x16px
- `favicon-32.png` - 32x32px
- `favicon.ico` - Multi-size ICO file (16, 32, 48px)

## Apple Touch Icons (iOS)
- `apple-touch-icon.png` - 180x180px (primary)
- `icon-120.png` - 120x120px
- `icon-152.png` - 152x152px

## PWA Icons (manifest.json)
- `icon-72.png` - 72x72px
- `icon-96.png` - 96x96px
- `icon-128.png` - 128x128px
- `icon-144.png` - 144x144px
- `icon-192.png` - 192x192px (also used as apple-touch-icon fallback)
- `icon-384.png` - 384x384px
- `icon-512.png` - 512x512px

## Windows Tiles (browserconfig.xml)
- `icon-70.png` - 70x70px
- `icon-150.png` - 150x150px
- `icon-310.png` - 310x310px

## Safari Pinned Tab
- `safari-pinned-tab.svg` - Monochrome SVG version of logo

## Tips
- Use a square version of your logo
- Ensure icons are legible at small sizes
- For PWA icons marked "maskable", add padding (safe zone is center 80%)
- Test with https://realfavicongenerator.net for comprehensive testing
